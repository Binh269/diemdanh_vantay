Futronic Standard SDK v4.2 for Windows 

1. To install the SDK, enter the folder 'setup', run the installation file 'sdk.4.2.2084.exe', after finished, the SDK example is installed to the folder 'C:\Program Files (x86)\Futronic\SDK 4.2\Examples\'.

2. For Java example, only x86 version 'ftrJSDK.dll' is provided in the setup. But we provide the source code of ftrJSDK, you can rebuild it for x64, or you can find it in the package 'other\SDK_4.2_Java_dll_x86_x64.rar'

3. For .Net example, please refer to the document 'doc\readme_net_example.pdf'.
